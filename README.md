# DSPACE - Democratized Space

You can find the game at [democratized.space](https://democratized.space).

Check out the [docs](https://democratized.space/docs)!

## Local Development

Clone and set up the project:

```bash
git clone https://github.com/democratizedspace/dspace.git
cd dspace
cd frontend
npm install
```

Start the development server:

```bash
npm run dev
```

Run tests:

```bash
npm test
```

### Code Quality

Before committing code, ensure your changes meet our code quality standards:

1. Check code formatting:
```bash
npm run format:check
```

2. Fix formatting issues:
```bash
npm run format:fix
```

The project uses:
- Prettier for code formatting
- ESLint for code quality
- EditorConfig for editor consistency
- Jest for testing
