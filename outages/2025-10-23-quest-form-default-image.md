# OUT-2025-10-23-quest-form-default-image

- **Date:** 2025-10-23
- **Component:** frontend quest creation
- **Root cause:** QuestForm validation continued to require a user-uploaded image even when the default quest image was injected, preventing the success banner from appearing in the manage-quests Playwright flow.
- **Resolution:** Allowed the autogenerated quest image to satisfy validation, normalized option state to keep empty item arrays for SSR, and added a regression test that submits the form with only the required fields populated.
