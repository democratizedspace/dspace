# Outage: E2E nav smoke failed on `/chat` due runtime build metadata 503

- **Date:** 2026-03-24
- **Component:** frontend/e2e-automation
- **Status:** Resolved

## Root cause

`frontend/scripts/setup-test-env.js` built preview artifacts but did not refresh
`frontend/src/generated/build_meta.json`. The `/build-meta.json` route therefore returned
`503` (`gitSha: missing`), and `/chat` logged a console resource failure that broke
`nav-route-smoke.spec.ts`.

## Resolution

- Updated setup-test-env to refresh build metadata (`resolveBuildMeta` + `writeBuildMeta`)
  before checking/rebuilding Astro artifacts.
- Added regression coverage to ensure setup-test-env keeps build metadata refresh before
  Astro build validation.
