# Toolbox

`/toolbox` is the utilities hub for maintenance and advanced workflows in v3.

## Typical use cases

- Jump to save safety tools before risky testing
- Access content editing/management helpers
- Reach diagnostics-oriented pages quickly

## Recommended workflow

1. Export a save backup before structural changes.
2. Use toolbox links to open editor/manage pages.
3. Validate behavior in `/stats` and relevant quest/process routes.
4. Use [Cloud Sync](/docs/cloud-sync) when moving across devices.
