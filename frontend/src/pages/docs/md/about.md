# About DSPACE

DSPACE is a local-first, incremental simulation game about learning practical systems that support
long-term space exploration. v3 focuses on grounded, reproducible skills: energy, fabrication,
automation, operations, and life-support workflows.

You progress by completing quests, running processes, and managing inventory while learning how
real-world systems interact.

## What v3 adds

v3 is the largest release so far and introduces:

- **10x quest growth** across official skill trees.
- **Custom content creation** for quests, items, and processes (plus in-game preview/validation
  tooling).
- **Local-first persistence** with IndexedDB primary storage, backups, and Cloud Sync support.
- **Expanded destination pages** (Stats, Titles, Toolbox, Settings, Leaderboard) that replaced
  earlier placeholders.
- **Persona-based NPC chat**, shipping with OpenAI in v3.0 while token.place is deferred to v3.1.

For release-by-release details, see the [Changelog](/docs/changelog) and
[v3 release state](/docs/v3-release-state).

## Core gameplay loop

1. Complete quests to unlock new systems and rewards.
2. Run processes to convert resources and craft progression items.
3. Balance dUSD (currency), dWatt (energy), and dCarbon (emissions pressure).
4. Build custom content and contribute it back through the submission workflow.

## Current scope (v3)

- Single-player progression is the primary experience.
- Guild and multiplayer mechanics are partially represented in social surfaces (for example,
  donation board visibility), but full ActivityPub-based multiplayer remains future roadmap work.
- The game is actively developed and receives regular content and quality updates.
