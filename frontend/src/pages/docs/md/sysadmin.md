Sysadmin skills focus on day-to-day system hygiene: working from the terminal, checking system
health, and responding to logs before issues become outages.

## What this skill covers

- Command-line basics for navigation and file management.
- Monitoring CPU, memory, storage, and network usage.
- Inspecting logs to spot errors and performance regressions.
- Establishing simple, repeatable maintenance routines.

## Recommended inventory

- A local or remote Linux host to practice commands.
- A logbook for tracking maintenance tasks and alerts.
- Monitoring dashboards or CLI tools like top, df, and journalctl.

## Quest highlights

- [Basic commands](/quests/sysadmin/basic-commands) builds terminal fluency.
- [Resource monitoring](/quests/sysadmin/resource-monitoring) tracks system load.
- [Log analysis](/quests/sysadmin/log-analysis) teaches incident triage.

## Best practices

1. Document every change and command sequence in a log.
2. Set alerts for disk space and memory pressure.
3. Practice log reviews on a schedule, not just during incidents.
4. Keep a maintenance checklist for updates and reboots.
