Sysadmin and DevOps quests cover the operational backbone of DSPACE: secure servers, reliable
backups, and repeatable deployments. These skills keep your infrastructure healthy while the
game scales.

## What you'll practice

- **CLI fundamentals** — navigate filesystems and read logs efficiently.
- **Service reliability** — monitor resources, rotate logs, and set up alerts.
- **Deployment workflow** — automate builds, ship containers, and manage clusters.
- **Security** — harden SSH, set firewall rules, and enable HTTPS.

## Quest path highlights

- Sysadmin basics: [Basic commands](/quests/sysadmin/basic-commands),
  [Log analysis](/quests/sysadmin/log-analysis),
  [Resource monitoring](/quests/sysadmin/resource-monitoring)
- DevOps flow: [Docker compose](/quests/devops/docker-compose),
  [CI pipeline](/quests/devops/ci-pipeline),
  [Enable HTTPS](/quests/devops/enable-https)
- Reliability: [Daily backups](/quests/devops/daily-backups),
  [Monitoring](/quests/devops/monitoring),
  [Firewall rules](/quests/devops/firewall-rules)

## Inventory and tools to expect

Operations inventory includes:

- Raspberry Pi hardware and cooling accessories
- SSDs and external backup drives
- Networking gear for secure deployments

## Operational habits

- Schedule backups before you need them, not after an incident.
- Keep a runbook of changes so you can roll back quickly.
