Robotics quests build on electronics fundamentals to create sensing, motion control, and navigation
skills for small robots.

## What you learn

- How to control servos and build simple actuators
- How to use sensors for line following, distance, and reflectance
- How to combine sensor input with motion planning

## Quest trailheads

- [Control a Servo Motor](/quests/robotics/servo-control)
- [Build a line-following robot](/quests/robotics/line-follower)
- [Measure distance with an ultrasonic sensor](/quests/robotics/ultrasonic-rangefinder)
- [Build a Servo Gripper](/quests/robotics/servo-gripper)
- [Add Obstacle Avoidance](/quests/robotics/obstacle-avoidance)
- [Navigate a Simple Maze](/quests/robotics/maze-navigation)

## Key gear

- Servo Motor and micro servo pair for motion testing
- Ultrasonic distance sensor and pan-tilt ultrasonic rig for navigation practice
- Two-servo arm kit or assembled servo arm for grasping exercises

## Common pitfalls

- Powering servos from logic pins instead of a stable supply
- Skipping calibration steps before obstacle-avoidance quests
- Letting wiring snag on moving parts during motion tests
