# Game Destinations (v3)

This page maps the major in-game destination routes that were promoted from placeholders to live
surfaces in v3.

## Core destinations

- **Stats** (`/stats`): progression overview for quests, items, and processes.
- **Titles** (`/titles`): unlockable rank roster and completion progress.
- **Leaderboard** (`/leaderboard`): donor/supporter standings.
- **Toolbox** (`/toolbox`): utilities hub linking save/content/diagnostic tools.
- **Settings** (`/settings`): preferences, auth token controls, migration helpers.

## Related utility destinations

- **Cloud Sync** (`/cloudsync`): GitHub gist backup upload/download flow.
- **Custom Content Backup** (`/contentbackup`): export/import local custom content.

## Why this matters

v3 navigation moved these routes into regular destination flows so players can access progress,
utilities, and account tooling without hidden or “coming soon” placeholders.
