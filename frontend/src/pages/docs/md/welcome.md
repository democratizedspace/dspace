# Welcome to DSPACE

DSPACE is a quest-driven simulation game where you learn systems by building them.

## Fast-start (v3)

1. Open [Quests](/quests) and complete the opening Welcome quests.
2. Use [Inventory](/inventory) and [Processes](/processes) to satisfy quest gates.
3. Track overall progress in [Stats](/stats), [Achievements](/achievements), and
   [Titles](/titles).

## Core gameplay loop

- **Quests** unlock progression, rewards, and new capabilities.
- **Items** are required/consumed/created by quests and processes.
- **Processes** convert resources over time.

This loop repeats across all quest trees.

## Pick your next tree

After onboarding, choose your next track from [Quest Trees](/docs/quest-trees):

- practical systems: Energy, Hydroponics, Electronics, Rocketry
- computing systems: Programming, DevOps, Sysadmin
- maker tracks: 3D Printing, Woodworking, Robotics

## v3 player tools you should know

- `/gamesaves`: manual game-state backup/restore
- `/contentbackup`: custom content backup/restore
- `/cloudsync`: optional encrypted gist backup flow
- `/toolbox`: diagnostics and utility shortcuts
- `/settings`: preferences, auth, migration utilities

## If you want to build content

v3 includes in-game creators:

- `/quests/create`
- `/inventory/create`
- `/processes/create`

Read [Content Development](/docs/content-development) and [Quest Submission](/docs/quest-submission)
for contribution workflows.
