# Leaderboard

Leaderboard (`/leaderboard`) is the in-game donation board.

## Current behavior (v3)

- Highlights top supporters in the Metaguild context
- Shows personal standing/progress indicators where available
- Acts as a social motivation layer without requiring multiplayer state sync

## What it is not

- Not a full guild ranking system yet
- Not a global real-time multiplayer scoreboard

For broader social plans, see [Guilds](/docs/guilds).
