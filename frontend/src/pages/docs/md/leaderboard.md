# Leaderboard

The Leaderboard page (`/leaderboard`) shows the Metaguild donation-board preview plus personal
contribution context.

## What is currently live

- ranked donor preview table
- personal contribution summary panel
- quest and energy leaderboard slices

This is an active social/progress surface in v3, while full guild mechanics remain deferred.

## Relationship to guilds

Leaderboard is live. Full guild systems (membership, shared guild state, guild-owned progression)
are not.

See [Guilds](/docs/guilds) for the current boundary.
