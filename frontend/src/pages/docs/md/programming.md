Programming quests focus on turning sensor data into insight. You will build small scripts,
visualizations, and web endpoints that make the rest of the system smarter.

## What this skill covers

- Reading sensor data and logging it to files.
- Calculating statistics like averages, medians, and standard deviation.
- Building JSON APIs and lightweight dashboards.
- Sending alerts or reports based on thresholds.

## Recommended inventory

- Temperature log CSVs and live JSON endpoints to practice analysis.
- A local machine or Raspberry Pi for running scripts and servers.
- Notes or logbooks to track API URLs and data formats.

## Quest highlights

- [Hello sensor](/quests/programming/hello-sensor) starts with raw readings.
- [Temp logger](/quests/programming/temp-logger) writes a persistent log.
- [JSON endpoint](/quests/programming/json-endpoint) publishes data on the network.
- [Graph temp](/quests/programming/graph-temp) and
  [Temp alert](/quests/programming/temp-alert) add visualization and automation.

## Best practices

1. Keep data formats consistent so charts and alerts stay reliable.
2. Version your scripts and include setup notes in comments.
3. Sanitize input and handle missing data gracefully.
4. Back up logs before running cleanup scripts.

## Where to go next

Programming skills power geothermal monitoring, robotics autonomy, and astronomy data analysis.
