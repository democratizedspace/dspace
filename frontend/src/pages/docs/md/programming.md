Programming quests teach you to collect data, expose it via simple APIs, and present it in useful
charts. The focus is practical telemetry for sensors and systems.

## What you learn

- How to log sensor readings and compute summary statistics
- How to build JSON endpoints and simple web dashboards
- How to alert when thresholds are exceeded

## Quest trailheads

- [Log Temperature Data](/quests/programming/temp-logger)
- [Serve Temperature as JSON](/quests/programming/temp-json-api)
- [Graph Temperature Logs](/quests/programming/graph-temp)
- [Serve a Web Page](/quests/programming/web-server)
- [Set Temperature Alert](/quests/programming/temp-alert)
- [Email Temperature Alert](/quests/programming/temp-email)

## Key gear

- Thermistor logging rig and temperature log CSV exports
- Live temperature JSON endpoint and live temperature dashboard output
- Temperature line chart and annotated temperature graph for reporting

## Common pitfalls

- Overwriting raw logs instead of appending with timestamps
- Building alerts without verifying sensor calibration first
- Forgetting to document API routes and refresh intervals
