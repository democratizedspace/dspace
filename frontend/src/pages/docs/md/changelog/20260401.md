Welcome to DSPACE v3. This release focuses on three big outcomes:

1. **Much deeper gameplay** (quests, items, and processes).
2. **Player-created content** with in-game tooling.
3. **More resilient architecture** for local-first play and backups.

If you are new to DSPACE, start with [About](/docs/about).

## v3 at a glance

Compared with v2.1 (the live baseline on `main`), v3 dramatically expands canonical content:

| Content type | v2.1 | v3 | Net new in v3 |
| --- | ---: | ---: | ---: |
| Official quests | 22 | 248 | **+226** |
| Built-in processes | 36 | 311 | **+275** |
| Built-in items | 90 | 507 | **+417** |

That means the quest catalog is now about **11.3× larger** than v2.1.

## Core player-facing upgrades

### 1) 10x+ quest expansion

- The v3 quest catalog now spans **248 quests across 19 trees**.
- Chemistry, Programming, and other newer trees are integrated into the main progression.
- Quest detail pages now render from canonical quest JSON, with clearer branching and reward visibility.

Related docs:

- [Quest Trees](/docs/quest-trees)
- [New Quests](/docs/new-quests)

### 2) Custom content creation in-game

v3 adds first-class creation workflows for player-made content:

- **Custom quests** at [Quest Submission Guide](/docs/quest-submission)
- **Custom items** at [/inventory/create](/inventory/create) (legacy alias: `/items/create`)
- **Custom processes** at [/processes/create](/processes/create)
- **Bundle submission flow** at [/bundles/submit](/bundles/submit) to keep related quests/items/processes together

Related docs:

- [Custom Content Bundles](/docs/custom-content-bundles)

### 3) Navigation and docs usability improvements

- The Docs area has improved structure and search behavior for faster discovery.
- `/docs/changelog` now has a dedicated landing experience.
- “More” navigation links behave correctly across subpath deployments.
- Settings, Stats, Titles, Toolbox, and Leaderboard are now active destinations.

### 4) Local-first saves + migration + backup

- v3 stores game state with a local-first architecture built around IndexedDB.
- Legacy save migration from earlier versions is supported.
- Cloud backup/sync is available via private gist integration.

Related docs:

- [Cloud Sync](/docs/cloud-sync)
- [Authentication](/docs/authentication)

### 5) AI chat and personas

- Chat now supports persona-driven interaction (for example Sydney, Nova, and Hydro).
- v3 production scope uses **OpenAI** as the active provider.
- token.place integration is implemented but intentionally deferred from v3 launch activation.

Related docs:

- [v3 Release State](/docs/v3-release-state)
- [token.place](/docs/token-place)

## Systems and economy updates

- Process execution now enforces requirements more consistently at start.
- Inventory and economy metadata were cleaned up so prices and currency semantics are clearer.
- Accessibility and responsive UI polish improved mobile workflows and keyboard/screen-reader clarity.

## Infrastructure and operations

- Continued migration to a self-hosted deployment model with stronger operational controls.
- Packaging, dependency, and CI reliability improvements reduced maintenance friction.
- QA automation and route/link validation coverage were expanded to reduce launch risk.

## Known launch-scope notes

- token.place is documented and integrated in code, but **not enabled as a production chat provider in v3.0 launch scope**.
- Completionist Award III is included in v3 launch scope, with reward behavior aligned to the award assembly flow.

## Looking forward

v3 is the largest DSPACE release so far and sets the foundation for faster content iteration.
Future releases will continue expanding social and collaboration features while keeping the core loop grounded in practical learning, simulation, and real-world mission alignment.

Thanks for being part of DSPACE.
