Welcome to DSPACE v3.

This release is focused on one goal: making the game deeper, easier to navigate, and easier to
contribute to without sacrificing reliability.

## v3 at a glance

Compared with v2.1 (the version currently running on `main` and at
democratized.space), v3 includes:

- **248 quests** across **19 quest trees** (**+226** vs v2.1)
- **311 processes** (**+275** vs v2.1)
- **507 items** (**+417** vs v2.1)

| Content area | v2.1 (`main`) | v3 | Net new in v3 |
| --- | ---: | ---: | ---: |
| Quests | 22 | 248 | +226 |
| Quest trees | 8 | 19 | +11 |
| Processes | 36 | 311 | +275 |
| Items | 90 | 507 | +417 |

The original “10x quests” goal is complete and exceeded: v3 ships at roughly **11.3x** the quest
count of v2.1.

## Major player-facing features

### 1) Content creation tools (no manual JSON editing required)

v3 introduces full in-game editors and contribution flows for:

- **Custom quests**
- **Custom items** (`/inventory/create`)
- **Custom processes** (`/processes/create`)
- **Unified bundle submissions** (`/bundles/submit`) so related quests/items/processes can be
  reviewed together

Documentation:

- [Quest Submission Guide](/docs/quest-submission)
- [Custom Content Bundles](/docs/custom-content-bundles)
- [Process Guidelines](/docs/process-guidelines)

### 2) Local-first saves + migration + backup

Progress is now stored primarily on-device (IndexedDB), with migration support for older saves and
safer backup workflows.

- Legacy data migration path from v2 storage
- Local backup/export workflows for custom content
- Optional private cloud backup via GitHub Gist

Documentation:

- [Cloud Sync](/docs/cloud-sync)
- [Authentication Flow](/docs/authentication)

### 3) Expanded navigation and utility surfaces

Previously placeholder or inactive destinations are now live and integrated, including:

- Stats
- Titles
- Toolbox
- Settings
- Leaderboard
- Improved docs/changelog navigation and filtering

### 4) AI chat personas with in-game context

v3 includes persona-based chat (for example Sydney, Nova, Hydro) and a gameplay-aware knowledge
context so chat responses can reference quests, inventory, and processes.

For provider status and rollout scope:

- [v3 Release State](/docs/v3-release-state)
- [token.place integration notes](/docs/token-place)

## Content expansion highlights

Beyond raw counts, v3 expands the playable learning paths and progression depth:

- Chemistry and Programming move from draft-era status into the main progression catalog
- Astronomy receives broader quest coverage, including meteor-shower-focused progression
- Rocketry guidance and docs are expanded for clearer launch prep and safety checks
- Guild and contribution docs are updated for active participation flows

Docs to explore:

- [Quest Trees](/docs/quest-trees)
- [Rockets](/docs/rockets)
- [Guilds](/docs/guilds)

## System and quality improvements

v3 also includes substantial quality and reliability work:

- Better route/nav coverage and docs route validation
- Expanded automated test coverage (unit, integration, E2E)
- Improved accessibility guardrails in UI linting and component behavior
- Process-start requirement enforcement improvements
- Infrastructure hardening for self-hosted deployment and CI consistency

## Notes for launch communications

If you are sharing this release with players, focus on these three points first:

1. **Depth**: 248 quests / 19 trees and significantly expanded item/process ecosystems.
2. **Creation**: players can now build and submit their own quests, items, and processes from the
   in-game UI.
3. **Reliability**: local-first saves, migration support, backup workflows, and stronger QA
   coverage.

Thanks for being here for v3 launch.
