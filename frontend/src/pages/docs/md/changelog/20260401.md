Welcome to DSPACE v3.

This release is focused on three things: significantly more playable content, creator tooling that
works in-game, and a more resilient local-first architecture for saves and custom content.

## v3 at a glance (vs v2.1)

Compared to v2.1 (the current `main` branch and production baseline), v3 expands the official
content catalog across quests, processes, and items.

| Catalog area | v2.1 | v3 | New in v3 |
| --- | ---: | ---: | ---: |
| Official quests | 22 | 248 | **+226** |
| Built-in processes | 36 | 255 | **+219** |
| Built-in items | 90 | 507 | **+417** |

That puts v3 at roughly **11.3×** the quest count of v2.1.

## Player-facing updates

### Navigation and destination upgrades

The navigation experience is now fully wired for real gameplay use:

- The More menu and header state stay accurate across subpath deployments.
- Dark mode is persistent and can be toggled from the site header.
- Placeholder links were replaced with working destinations, including:
    - [Stats](/stats)
    - [Titles](/titles)
    - [Toolbox](/toolbox)
    - [Leaderboard](/leaderboard)
    - [Settings](/settings)
- The docs landing page now uses a structured manifest and fast filtering for easier discovery.
- [/docs/changelog](/docs/changelog) now has a proper release-notes landing page.

### Major content expansion

v3 ships the largest official content drop in DSPACE so far:

- Official quests now span **19 quest trees**.
- Chemistry and Programming are now part of the main progression.
- Astronomy has expanded launch content (including meteor-shower observation flow).
- Completionist Award III is included in v3 launch scope.

For full tree-level breakdowns, start with:

- [Quest Trees](/docs/quest-trees)
- [New Quests](/docs/new-quests)

### Economy and gameplay data polish

v3 removes many placeholder values and improves economy consistency:

- First-aid consumables now include concrete pricing.
- Hydroponics pricing and soulbound behavior are clarified where needed.
- Currency items (dCarbon, dWatt, dUSD) are represented consistently as currency inventory items.

## Creator tooling: now first-class in-game

v3 introduces a full creator workflow for community content:

- Custom quest creation in-app with validation and preview support.
- Custom item and process editors with schema-backed checks.
- Bundle submission workflow for packaging related quests, items, and processes together.

Start here:

- [Quest Submission Guide](/docs/quest-submission)
- [Custom Content Bundles](/docs/custom-content-bundles)
- [Content Development](/docs/content-development)

## Local-first saves, backups, and migration

v3 moves saves and custom content to a local-first model:

- Legacy v2.1 save migration is handled on first launch.
- IndexedDB becomes the primary storage layer for better durability.
- Cloud Sync provides encrypted, user-controlled backup/restore workflows.

Related docs:

- [Legacy Save Storage](/docs/legacy-save-storage)
- [Cloud Sync](/docs/cloud-sync)

## AI chat and persona updates

NPC chat is now live with selectable personas (for example Sydney, Nova, and Hydro), plus improved
in-game knowledge grounding from quests, items, processes, and local inventory context.

v3 production scope is **OpenAI chat**. token.place integration is present in code and tracked for
post-launch rollout.

See:

- [v3 Release State](/docs/v3-release-state)
- [token.place](/docs/token-place)

## Infrastructure and quality improvements

- Broader Playwright coverage for navigation, docs routing, and mobile layout behavior.
- Additional safeguards for process gating, accessibility checks, and route reliability.
- Helm persistence defaults updated for smoother k3s storage-class behavior.

## Known limitation at launch

- Chat provider scope in v3 is OpenAI-only at launch; token.place activation is deferred to the
  v3.1 follow-up lane.

## Thank you

Thank you for building and testing DSPACE with us.

v3 is a major foundation release: much larger content coverage today, better systems for creators,
and stronger platform reliability for the next set of launches.
