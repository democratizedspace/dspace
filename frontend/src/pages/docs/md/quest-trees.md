# Quest Trees

DSPACE v3 organizes quests into themed trees so players can specialize while still following a
clear progression path.

## Active official trees

- **3D Printing**
- **Aquaria**
- **Astronomy**
- **Chemistry**
- **Completionist**
- **Composting**
- **DevOps**
- **Electronics**
- **Energy**
- **First Aid**
- **Geothermal**
- **Hydroponics**
- **Programming**
- **Robotics**
- **Rocketry**
- **Sysadmin**
- **UBI**
- **Welcome**
- **Woodworking**

Chemistry and Programming are now treated as fully integrated progression trees in v3.

## How to use this page

- For player-facing guidance, open each skill doc from the main [/docs](/docs) index.
- For contribution/QA work, treat each tree as coupled with a matching docs page under
  `frontend/src/pages/docs/md/<tree>.md`.

## Contributor integrity rules (summary)

When editing quest JSON for a tree, update that tree's docs in the same PR so rewards, gates,
and process dependencies stay aligned.
