# Stats

The Stats page (`/stats`) is a live v3 dashboard for progress tracking.

## What it shows

- Quest completion totals
- Inventory breadth and item accumulation context
- Process activity summaries
- Snapshot-style progression metrics useful for QA and personal goals

## Why use it

- Validate your save after imports or migrations
- Check whether you are gating on quests, resources, or processes
- Track milestone momentum without opening each subsystem page separately

Stats read from your local save state and update as gameplay changes.
