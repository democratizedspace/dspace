# Stats

The Stats page (`/stats`) reports live catalog metrics derived from in-repo quests, items, and
processes.

## Current metrics include

- total quests and quest categories
- dialogue node and option totals
- unique NPC guide count
- number of quests with item rewards
- number of quests that launch processes
- inventory totals by category
- total processes and process-duration coverage

## What Stats is for

- sanity-checking content growth between releases
- quick QA confirmation that expected catalog sizes are present
- contributor visibility into progression breadth
