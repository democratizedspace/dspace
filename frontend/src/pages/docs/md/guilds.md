# Guilds in v3

Guilds are partially represented in v3, with full gameplay still planned.

## What exists now

- A Guilds destination (`/guilds`) in navigation context.
- Metaguild framing in docs and social progression language.
- Leaderboard integration (`/leaderboard`) for donor/community recognition.

## What is not fully live yet

- persistent guild membership gameplay loops
- shared guild inventories/resources
- cooperative guild quest state
- federated/activitypub guild operations

## Practical guidance for players

- Use `/leaderboard` for current social visibility.
- Treat `/guilds` as current-state + roadmap framing, not a completed gameplay loop.
- Check [Roadmap](/docs/roadmap) and [v3 Release State](/docs/v3-release-state) for rollout status.
