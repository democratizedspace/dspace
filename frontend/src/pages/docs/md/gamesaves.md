# Gamesaves

`/gamesaves` provides export/import tooling for full game-state backups.

## Page capabilities

- export your current save snapshot
- copy a portable backup payload
- import a previous backup payload to restore state

This route is the primary manual safety net before large gameplay or custom-content experiments.

## Good backup habits

- export before major quest chain progress
- export before importing experimental content
- keep timestamped copies in a private location
- verify restore success by checking quests, inventory, and processes
