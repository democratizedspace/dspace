# Testing Guide

This guide covers the expectations for running tests in DSPACE and where to find
deeper documentation.

## What to run before a PR

- `npm run lint`
- `npm run type-check`
- `npm run build`
- `npm run test:ci`
- `npm run audit:ci`

## Detailed references

For the full testing walkthrough, including component patterns and debugging tips,
see the repository Testing Guide at
https://github.com/democratizedspace/dspace/blob/v3/frontend/TESTING.md.
